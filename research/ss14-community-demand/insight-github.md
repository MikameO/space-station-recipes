# GitHub feature-request demand — space-wizards/space-station-14

Источник — трекер задач основного апстрима. Это самый **product-adjacent** канал из всех: precomputed theme-hits дают chemistry 19, medical 11, cooking 9, botany 5, science 5 при atmos 14 и antag 20, а доминирует ui_qol (63). То есть химия+медицина вместе (30 упоминаний) — второй по величине смысловой кластер после общего UI/QoL и наравне с antag. Но абсолютная "громкость" здесь низкая: медиана +2, потолок +18. Поэтому продуктовая ценность источника — не в силе голосов, а в том, что он **точно называет болевые точки систем**, которые моделирует наш справочник (реагенты, реакции, метаболизм, крафт-деревья, ботаника), и показывает, куда движется схема данных апстрима.

Ниже инсайты сгруппированы: сначала **продуктово-релевантные** (gh-01…gh-10, приоритет), затем **общий спрос как контекст** (gh-11…gh-20).

## Калибровка demand_strength для этого источника
- **strong** — plus1 ≥ 8 (топ ~5%, всего 9 задач) ИЛИ аномально высокая дискуссия (comments ≥ 20).
- **moderate** — plus1 4–7 (верхний квартиль, ~37 задач) ИЛИ заметная дискуссия (comments ~8–15).
- **weak** — plus1 ≤ 3 (большинство, 159 из 196). Внутри продуктового подмножества потолок = +6, поэтому почти весь наш профильный спрос попадает в "moderate/weak" — это ожидаемо для нишевых департаментов.
